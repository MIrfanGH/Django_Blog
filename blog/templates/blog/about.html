About This Project

This project is a feature-rich blogging platform built using Django and deployed on AWS. It goes beyond basic CRUD functionality and incorporates performance optimization, background processing, caching strategies, and AI-powered features

The project demonstrates real-world web development practices including database design, cloud deployment, security best practices.

Key Features

  • Full user authentication with email-based password reset
  • Role-based access control (author and admin permissions)
  • CRUD-enabled blog publishing with pagination and optimized queries
  • Redis caching with signal-driven cache invalidation
  • Celery-based asynchronous background task processing
  • Stripe Checkout–based donation system with webhook event handling
  • On-demand AI-powered blog summarization using LLMs
  • Non-destructive AI summaries that preserve original content
  • AWS S3 for media and static asset storage
  • Production deployment using Nginx and Gunicorn on AWS EC2

    • Planned Enhancements
  • Infrastructure autoscaling using Terraform

Technology Stack

  • Django (Class-Based Views, Signals, Authentication)
  • PostgreSQL (relational database via Django ORM, AWS RDS)
  • Redis (caching and message broker)
  • Celery (asynchronous task processing)
  • Stripe API (Checkout Sessions & Webhooks)
  • LLM APIs for AI-powered content summarization
  • AWS EC2 (application hosting)
  • AWS S3 (media and static asset storage)
  • Cloudflare (DNS, CDN, SSL, DDoS protection)
  • Nginx (reverse proxy and static file handling)
  • Gunicorn (production WSGI server)
